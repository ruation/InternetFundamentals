# Servers and Hosting - Summary

A **server** is a computer (or system) that provides resources, services, or data to other computers over a network. On the Internet, servers are responsible for storing websites, handling requests, and sending responses to clients (such as browsers).

When you access a website, your browser sends a request to a server, and the server responds with the webpage content.

## What a Server Does

Servers can perform many functions, such as:

- Hosting websites and web applications
- Storing and managing files and databases
- Processing user requests
- Handling authentication (logins)
- Serving APIs and backend services

Servers usually run 24/7 and are designed to handle many users at the same time.

## Web Server

A **web server** is a server that handles HTTP/HTTPS requests. Popular web server software includes:

- Apache
- Nginx
- Microsoft IIS

A web server receives requests from browsers and sends back responses, usually containing HTML, CSS, JavaScript, images, or other resources.

## What Hosting Means

**Hosting** is the service of providing a place on a server where a website or application can run and be accessed through the Internet.

Hosting companies maintain the physical servers, network connection, and infrastructure, while users upload their websites and configure them to go online.

## Types of Hosting

### Shared Hosting
- Many websites share the same server resources.
- Cheaper, but slower and less flexible.

### VPS (Virtual Private Server)
- A physical server is divided into multiple virtual servers.
- More control and better performance than shared hosting.

### Dedicated Server
- One customer rents an entire physical server.
- High performance, but more expensive.

### Cloud Hosting
- Uses multiple servers working together.
- Scalable and reliable, often used for modern applications.

## Domain and Hosting Relationship

To make a website available online, you usually need:

- A **domain name** (example: `mywebsite.com`)
- A **hosting service** (where the website files are stored)
- DNS configuration to connect the domain to the server’s IP address

## Key Takeaways

- A server provides services and data to other devices over the Internet.
- Hosting is a service that allows websites to be stored and accessed online.
- Different hosting types offer different levels of performance, cost, and control.
- A domain name must be connected to hosting using DNS.
