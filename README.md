# 🌐 How the Internet Works

This repository documents my learning journey about how the internet works, from basic concepts to more advanced topics.

## 📌 About

The goal of this project is to understand the fundamental concepts behind the internet, including how data travels, how websites work, and how different technologies interact.

To start, read the ["what-is-the-internet"](fundamentals/what-is-the-internet.md) documentation.

## 📚 Topics Covered

- What is the Internet?
- How data travels (packets, routing)
- IP addresses and TCP
- DNS
- HTTP
- How browsers work
- Servers and hosting

## 🎯 Goals

- Improve my understanding of networking concepts
- Practice explaining technical topics in English
- Build a solid foundation for web development
- Create a useful reference for future studies
